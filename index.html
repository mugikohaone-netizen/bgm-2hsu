Audio → Embeddable Code

BGM→コード変換ツール

音楽ファイルをHTMLアプリに埋め込めるコードに変換します。ループ範囲を指定すれば、イントロ付きのBGMもシームレスにループ再生させるコードが作れます。

使い方　① 音楽ファイルを選択 → ② 波形でループ範囲を確認・調整 → ③ コードをコピー → ④ HTMLに貼り付け
♪
音楽ファイルをドロップ、またはタップして選択
mp3 / wav / ogg・複数選択可

共通コード（ページに1回だけ貼り付け）


    

  


  


  

    ※ ループ範囲の指定は「再生位置の制御」であり、それ単独では埋め込みデータのサイズは小さくなりません。サイズを抑えたい場合は各ファイルの「サイズを小さくする」から圧縮してください（MP3のまま再エンコードするので再生互換性はそのままです）。

    ※ 目安として、base64化すると元ファイルよりおよそ33%大きくなります。
